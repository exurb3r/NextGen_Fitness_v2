# NextGen Fitness

> **A Modern Web-Based Gym Membership Management System**

---


**NextGen Fitness** is an undergraduate capstone project that aims to modernize gym management while promoting member engagement through fitness tracking, gamification, and social features.

This project is a complete rebuild of our previous prototype. Instead of continuing with the original MERN implementation, the system is being redesigned using a more scalable architecture, stronger type safety, and modern web development practices.

The goal is not only to improve the user experience but also to create a cleaner, maintainable, and production-oriented codebase suitable for long-term development.

---

## Reason

Our initial prototype successfully validated the project's concept, but as development progressed, several architectural limitations became apparent.

### Previous Challenges

- Inconsistent file organization
- Large and tightly coupled components
- Limited scalability
- Weak type safety
- Difficult maintenance as the project grew

Rather than continuously patching the existing codebase, we decided to rebuild the system with a more structured architecture while preserving the original project vision.

---

## 🎯 Project Goals

The system aims to provide a centralized platform for both gym administrators and members.

## Development Philosophy

This version leans more on building a cleaner, scalable, and maintainable system.

### Principles

- Feature-based architecture
- Separation of concerns
- Strong typing with TypeScript
- Reusable UI components
- Consistent folder structure
- Scalable state management
- Input validation
- Modular API design
- Professional coding practices

## Project Status

>  **Active Development**

This project is currently being rebuilt from the ground up as part of our undergraduate capstone project.

The current focus is establishing a scalable architecture, improving code quality, and creating a maintainable foundation before implementing the complete feature set.
